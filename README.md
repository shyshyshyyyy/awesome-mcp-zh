# Awesome MCP 服务器与 Claude Skills（中文） [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> 精选的 MCP（Model Context Protocol）服务器与 Claude Skills 中文清单 —— 按场景分类，**每个条目附中文说明、安装命令与客户端配置**，可一键接入 Claude、Cursor 等。

**MCP** 是 Anthropic 推出的标准化协议，让 AI 模型能安全、高效地访问外部工具与数据源（搜索、数据库、文件、浏览器自动化等）。**Claude Skills** 则是让 Claude 具备特定能力（生成 PDF、Word、PPT、Excel 等）的技能包。

完整目录（持续更新，含安装配置）👉 **[嘉名AI导航 · MCP 导航](https://nav.jiamingai.cn/mcp)**
